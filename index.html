What does “coding for kids” mean?

Coding for kids refers to the collection of opportunities available for children to get involved in coding. As mentioned, while it might be difficult to imagine a young mind learning something so seemingly complex, it's definitely an attainable reality.

Thanks to the proliferation of many coding summer camps, programs, websites, and toys, coding can be fun and digestible. On the most basic level, coding is how we communicate with computers, and what we use to build and run websites, apps, video games, and more. Learning to code is like learning how to speak and write in a particular language; a computer’s language.

Years ago when all of this kids and code chatter started, you could have characterized it has hype because the whole idea was new and novel to the education system. And, while this “learn to code” popularity spike wasn’t unfounded by any means, time was really the only thing that could tell us if it all was going to be a big fat flash in the pan.
